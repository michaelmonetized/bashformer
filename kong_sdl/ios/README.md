# Platformer SDL - iOS Version

iOS version of the Platformer SDL game built with Swift and SpriteKit.

## Requirements

- Xcode 14.0 or later
- iOS 15.0 or later
- Swift 5.5 or later

## Building

1. Open `PlatformerSDL.xcodeproj` in Xcode
2. Select your target device or simulator
3. Build and run (âŒ˜R)

## Controls

- Tap left side of screen: Move left
- Tap right side of screen: Move right
- Swipe up: Jump
- Tap center: Attack (sword)
- Button controls available in UI
