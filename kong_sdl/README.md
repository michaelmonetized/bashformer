# Platformer SDL - Mobile Ports

This directory contains the Platformer SDL game ported to iOS and Android.

## Structure

- `shared/` - Shared C code (platformer_sdl.c)
- `ios/` - iOS Swift version using SpriteKit
- `android/` - Android native C version with JNI wrapper

## Building

### iOS
See `ios/README.md` for instructions on building with Xcode.

### Android
See `android/README.md` for instructions on building with Android Studio/NDK.
